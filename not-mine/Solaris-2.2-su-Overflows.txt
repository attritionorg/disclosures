Buffer Overflow in Environment Variable Concatenation (Lines 170-177)
C
strcat(homedir, dir);           // homedir is 64 bytes: "HOME="
strcat(logname, name);          // logname is 20 bytes: "LOGNAME="
strcat(termtyp, term);          // termtyp is 20 bytes: "TERM="
strcat(hzname, hz);             // hzname is 10 bytes: "HZ="
strcat(tznam, tz);              // tznam is 15 bytes: "TZ="
Issue: These buffers are severely undersized. A user with a long home directory, username, or environment variable can overflow.

Impact: Stack buffer overflow → arbitrary code execution.

Fix: Use dynamic allocation or snprintf() with size checks.
